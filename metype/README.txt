Metype is an engagement engine from the future. Audience will be able to communicate, interact and have conversations in realtime and publishers have the right tools to moderate and filter the content.

== Description ==
Metype is a commenting system with realtime conversations and live updates. This lets build user communities, retain audience and grow user engagement. This is like a mini social network on your site.

For more details visit: [Metype](https://www.metype.com)

== Installation ==

1. Go to **"Plugins"** > **"Add Plugin"**
2. Search for **"Metype Engagement Engine"** by Metype
3. Click **"Install Now"**
4. Activate the plugin
5. Go to [Metype](https://www.metype.com). Click on **"Get started"** > **"Sign up"** 
6. Once you reach admin panel. Click on Embed metype. Under Metype for wordpress. Copy the account-id.
7. Go to wordpress dashboard. Click on metype on sidebar menu. Enter the account-id in the empty box and click save. Metype is live on your site.
